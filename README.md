
# Changelog

## [1.0.0] — 2026-06-16

- Добавлено десктопное приложение для ПК на C# WPF.
- Добавлено мобильное приложение для Android на .NET MAUI.
- Подключена облачная база данных Supabase.
- Реализована авторизация пользователей через Supabase Auth.
- Добавлены роли пользователей: администратор, оператор, специалист.
- Реализовано разграничение доступа к данным по ролям.
- Добавлена таблица профилей пользователей.
- Добавлена основная таблица обращений.
- Добавлены справочники: типы обращений, статусы обращений, приоритеты, роли, видимость комментариев, служебные метки.
- Реализована структура базы данных в третьей нормальной форме.
- Добавлены SQL-скрипты: `variant17_schema.sql`, `variant17_seed.sql`, `variant17_supabase_rls.sql`, `variant17_after_auth_link_TEMPLATE.sql`.
- Добавлены тестовые записи обращений в базу данных.
- Реализован список обращений в ПК-приложении.
- Реализован список обращений в мобильном приложении.
- Реализована карточка обращения.
- Реализовано добавление обращения.
- Реализовано редактирование обращения.
- Реализовано удаление обращения через мягкое удаление.
- Реализована смена статуса обращения.
- Реализованы комментарии к обращениям.
- Реализованы внутренние и внешние комментарии.
- Реализовано скрытие внутренних служебных данных от мобильного приложения.
- Реализованы поиск, фильтрация и сортировка обращений.
- Реализованы уведомления о высоком приоритете.
- Реализована статистика по обращениям.
- Реализован отдельный экран статистики в мобильном приложении.
- Реализованы графики по статусам и приоритетам.
- Реализован экспорт отчетов в Word и Excel.
- Реализован журнал действий и логирование событий.
- Добавлена локальная очередь синхронизации.
- Добавлен локальный кэш для работы при временном отсутствии сети.
- Добавлена стратегия разрешения конфликтов LWW.
- Добавлено обновление данных после восстановления подключения.
- Добавлена нижняя навигация в мобильном приложении.
- Добавлены иконки нижней навигации.
- Добавлена фирменная иконка приложения.
- Добавлен splash screen.
- Добавлено оформление в стиле Ростелекома.
- Добавлена светлая и темная смена фона приложения.

### Изменено
- Firebase заменен на Supabase как основная облачная база данных.
- Тексты и функции адаптированы под тему: учет обращений граждан и клиентов.
- Исправлены надписи в схемах и отчете, связанные с Firebase.
- Обновлена логика подключения к облачной базе через REST API Supabase.
- Обновлен интерфейс мобильного приложения: улучшены карточки обращений, исправлены кнопки быстрых действий, улучшена читаемость текста.
- Желтые плашки заменены на фиолетовые.
- Логика смены темы изменена так, чтобы менялся фон приложения, а не цвета кнопок.
- Смена фона сделана глобальной для всех вкладок мобильного приложения.
- Улучшена структура вкладок мобильного приложения: обращения, статистика, еще.

### Исправлено
- Исправлена ошибка `CS8978`, связанная с generic-методами и nullable-типами.
- Исправлена ошибка `MC3074` в XAML-разметке.
- Исправлены ошибки `CS0246` с типом `RoundRectangle`.
- Исправлена ошибка `CS0120` с использованием `Grid.SetRow`.
- Исправлена ошибка отсутствия `MainLauncher` в Android-проекте.
- Исправлена проблема, когда Visual Studio не видела запускаемое действие Android.
- Исправлены поля авторизации, которые были плохо видны.
- Исправлено обрезание кнопок в мобильном интерфейсе.
- Исправлена проблема, когда кнопка смены фона переставала работать после перехода между вкладками.
- Исправлена логика отображения черного текста на светлых элементах.
- Исправлена навигация между экранами мобильного приложения.
- Исправлены подписи и схемы в отчете под Supabase.

### Технические сведения
- Язык разработки: C#.
- ПК-приложение: WPF.
- Мобильное приложение: .NET MAUI Android.
- Облачная база данных: Supabase/PostgreSQL.
- Авторизация: Supabase Auth.
- Синхронизация: REST API + локальная очередь операций.
- Целевые платформы: Windows и Android.
